## 三、规则九：酒店多业态邻近性与防噪缓冲矩阵 (Adjacency Matrix)

在高层商务酒店设计中，功能流线交叉和声学干扰是两个致命痛点。在生成 XYZ 坐标前，AI 控制器必须无条件通过以下拓扑过滤。

### 空间流线及防噪声拓扑隔离图

```
[ 26F 云顶酒吧/行政酒廊 ] ── 高空景观
          │
[ 16F - 25F 行政客房 ]
          │
[  6F - 15F 标准客房 ]
          │
=============[ 5F 办公层：声学物理隔离缓冲层 ]=============    ← 防御对象: 4F宴会厅扩声
          │
[  4F 宴会厅/会议区 ]        ── 大跨度空间、强扩声（婚宴/活动）
          │
[  3F 餐饮/厨房 ]            ── 厨房热厨排烟、传菜流线
          │
[  2F KTV/水疗会所 ]         ── [★强噪声源] 超低音、振动
          │
[  1F 酒店大堂/商铺 ]        ── 客流总枢纽（四流线分流）
          │
=============[ B1F 地下车库/设备房 ]=============
```

### 3.1 强吸引力法则 (Strong Attraction - 紧邻或共用通道)

- **餐厨紧邻（Kitchen-Dining Boundary）**：[3F 中餐/全日餐厅] 必须与 [3F 主厨房] 水平紧邻（通行距离 ≤ 15m）；[4F 大宴会厅] 必须与 [4F 宴会配餐间] 通过专用服务内廊/传菜梯直连
- **大堂延伸（Lobby-Lounge Boundary）**：[1F 大堂吧] 必须位于 [1F 酒店大堂] 的视线黄金延伸区，空间上无硬性物理分隔，进深不大于 10m
- **大堂核心筒直达（Lobby-Core Boundary）**：[1F 酒店大堂] 必须与客梯核心筒直连，且从前台到客梯厅的视线不被柱子或隔墙遮挡

### 3.2 强排斥力法则 (Strong Repulsion - 物理与垂直隔离)

- **KTV与客房垂直抗噪隔离（Acoustic Buffer Zone）**：[2F KTV会所] 属于极强振动与噪声源，严禁与 [16F+ 行政客房] 垂直贴临。硬约束：2F KTV 与 16F 行政客房之间必须保留完整的 3F 餐饮 + 4F 宴会 + 5-15F 办公楼层作为楼板衰减路径。所有 KTV 箱体底部必须设 ≥ 100mm 浮筑隔声层
- **宴会厅扩声与客房抗噪隔离（Banquet Buffer Zone）**：[4F 宴会厅] 举办大型活动时产生强扩声与低频振动。硬约束：必须保留 [5F 办公层] 作为阻尼缓冲层。4F 宴会厅大跨度空间的结构转换梁必须在 5F 办公层完成，不可直接转换到 6F 客房层
- **大型设备振动的垂直退让（MEP Isolation）**：[B1_MEP 大型集中机房] 内包含制冷主机和变压器，严禁布置在 [1F 酒店大堂] 主前台及 [4F 大宴会厅] 的垂直投影正下方，避免结构传声
- **客货绝对流线隔离（Circulation Separation）**：[1F 酒店大堂主入口] 与 [B1_BOH 后勤货运卸货平台] 必须布置在建筑场地的相反朝向。四种流线的空间映射关系如下：
