# Rhino MCP 生成规则 · 铁律清单

## 规则一：单位铁律
所有 Rhino 坐标值必须使用 **毫米（mm）**，不允许出现未经 ×1000 处理的米制数字。

- 1m = 1000mm
- 场地 105m × 90m = 105000 × 90000
- 层高 4.2m = 4200
- Z 轴起始标高总是从 0 开始

## 规则二：面积铁律
每次生成前必须先算面积再建模。目标 **37,807㎡（±5%）**。校核逻辑：

```
裙房基底 × 4层 + 办公 × 15层 + 酒店 × 26层 ≈ 37,807㎡
模板尺寸：裙房 ~2,900㎡/层, 办公 ~600㎡/层, 酒店 ~625㎡/层
```

如果面积不对，必须先调尺寸再生成，不允许"先做再说"。

## 规则二B：不超建筑红线铁律
所有体块必须完全位于退线范围（建筑控制线）之内。退线要求：东≥15m、西≥10m、南北≥8m。

可建范围坐标：X:[15000, 95000] × Y:[8000, 82000]
- 任何体块的 XY 坐标不得超出此范围
- 每次生成前必须校验所有体块的 X 方向、Y 方向最小/最大值
- 如果超出必须先调尺寸，不允许"做完再移"

## 规则三：塔楼铁律
塔楼必须从裙房顶部升起，不允许悬空：

- 办公塔楼 1F 底部标高 = 裙房顶 = H×4
- 酒店塔楼 1F 底部标高 = 裙房顶 = H×4
- 塔楼下方必须有裙房体块填充整个底面

## 规则四：规范铁律（JGJ62-2014）
| 项目 | 要求 |
|:----|:-----|
| 客房净面积 | ≥25㎡ |
| 走廊净宽 | ≥1.5m |
| 无障碍客房 | 2间 |
| 车库层高 | 3.6m |
| 密度 | ≤35% |
| 容积率 | ≤4.0 |
| 限高 | ≤99.3m |

## 规则五：布局差异铁律
ABC 三个方案必须有明显不同的布局逻辑，不允许相同。

- **A** 围合内院式：裙房 L 形，围合出中央庭院
- **B** 线性退台式：裙房沿主干道线性展开，酒店 16F 以上退台
- **C** 集中裙房式：裙房大底盘居中，双塔对称布局

## 规则六：图层命名规范
```
[方案字母 A/B/C]
├── 控制线
├── 裙房
│   ├── 1F → 大堂 / 商铺 / 宴会门厅
│   ├── 2F → KTV / SPA
│   ├── 3F → 全日制餐厅 / 中餐厅 / 厨房
│   ├── 4F → 宴会厅 / 会议室
│   └── 核心筒
├── 办公塔楼
│   ├── 1F-15F → 办公区
│   └── 核心筒
└── 酒店塔楼
    ├── 1F-15F → 设备段 (灰色)
    ├── 16F → 麻将客房 (紫色)
    ├── 17F-26F → 客房 (橙色/浅橙交替)
    └── 核心筒
```

## 规则七：方案标注规范
每个方案必须标注名称，用 TextDot 放在场地右上角外侧：
```
A 围合内院式
B 线性退台式
C 集中裙房式
```
